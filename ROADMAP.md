## 優先度サマリー

### 高 (次に着手)
1. `Ctrl+E` / `Ctrl+Y` — 1行スクロール
2. `ZZ` / `ZQ` — ショートカット終了
3. Insert `Ctrl+W` / `Ctrl+U` / `Ctrl+O`
4. `:noh` — 検索ハイライトクリア
5. `:registers` / `:marks` 表示
6. `'<` / `'>` マーク
7. Quickfix ウィンドウ UI (`copen` のリスト表示)
8. grep/vimgrep 結果を Quickfix に流し込む
9. ブランチ名のステータスバー表示
10. ハンクのステージング
11. タブの未保存インジケーター
12. ファイルエクスプローラのキーボード操作
13. `:s` のリアルタイムプレビュー (`inccommand`)
14. ターミナル ↔ エディタのフォーカス移動

### 中 (次のフェーズ)
- `(` / `)` センテンスモーション
- `gE`, `gn` / `gN`, `[p` / `]p`
- VisualBlock `$` / `~`
- フォールド: `zj`/`zk`/`[z`/`]z`/`zd`
- `:cd`, `:pwd`, `:echo`, `:execute`, `:unmap`
- `@@` マクロ再実行
- `autocmd` / `viminfo` 永続化
- LSP `documentHighlight`
- `:Git status`、コンフリクトマーカーハイライト
- コマンドパレット、プロジェクト設定 (.editorconfig)

### 低 (将来)
- Vimscript (`:function`, `:if`, `:for`)
- DAP デバッガ統合
- リモート編集
- ブランチングアンドゥ
- プラグインシステム
- ワークスペース (マルチルート)
